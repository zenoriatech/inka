# Whisperer

Private, on-device dictation for macOS. Hold right ⌘, speak, release — text appears at the cursor. All recognition runs on your Mac; audio never leaves the machine.

**Requirements:** Apple Silicon (M1 or later), macOS 14+.

## Download

Get the latest build from [Releases](https://github.com/zenoriatech/whisperer/releases/latest).

This repository hosts signed, notarized builds for testing. It contains no source code.
